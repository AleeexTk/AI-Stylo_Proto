# AI-Stylo Firebase (Autonomous Project)

This directory contains the autonomous, cloud-native version of AI-Stylo, built to leverage the Firebase ecosystem (Firestore, Authentication, Hosting, and Cloud Functions).

## Architecture Approach

- **Frontend**: React-based Style Advisor with hybrid PEAR execution.
- **Backend / DB**: Firebase (Cloud Functions + Firestore Vector Search).
- **AI Integration**: Dual Gemini integration (Cloud-to-Cloud and Browser-to-Cloud).
