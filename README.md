# AI-Stylo_Proto launcher wrapper

Этот уровень репозитория содержит обёртки запуска для случая, когда проект находится во вложенной папке `AI-Stylo-main` (типично после ZIP-распаковки).

## Запуск в 1 клик из корня

- Windows: `start_ai_stylo.bat`
- macOS/Linux: `./start_ai_stylo.sh`

Скрипты автоматически находят `AI-Stylo-main/launcher.py` (или `launcher.py` в текущем каталоге) и запускают его.
