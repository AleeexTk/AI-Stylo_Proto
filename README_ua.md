# 🧬 AI-Stylo Core: Персональна Fashion ОС

## AI-Движок підбору образів та інтеграційний шар гейміфікації

AI-Stylo — це система, побудована за парадигмою **EvoPyramid "Архітектура Архітектур"**.
Вона забезпечує надійне ядро для персоналізації моди, жорстко відокремлюючи бізнес-логіку (Контракти, Ембеддинги, Телеметрію, Гейміфікацію) від візуального шару представлення (Вершини).

## 🚀 Два режими відображення (Вершини) на одному ядрі

З AI-Stylo один бекенд керує абсолютно різними користувацькими досвідами:

1. **Режим RPG-Інвентаря (`apps/web/streamlit_rpg/app.py`)** — Захопливий гейміфікований дашборд. Користувачі збирають "образи" у слотах екіпіровки, прокачують динамічні навички на основі своїх дій та досліджують моду як гру.
2. **Режим B2B Плагіну (`apps/web/streamlit_b2b/app.py`)** — Легкий віджет "Згенерувати образ" для інтеграції безпосередньо в картки товарів партнерських магазинів, що миттєво підвищує крос-сейли.

## 🏗 Системні інваріанти (Ядро)

* **Контракти (`apps/core/contracts.py`)**: Строгі класи даних (`Item`, `Profile`, `Outfit`, `Event`), що гарантують відсутність хаосу в управлінні станами.
* **Телеметрія та Движок Навичок (`apps/core/skills_engine.py`)**: Розумний аналізатор потоку подій. Відкриває скіли ("Постійність смаку", "Фокус на вигоді") лише після їх досягнення — жодних демотивуючих "заблокованих" нулів.
* **Партнерські Пакети (Overrides)**: Ритейлери можуть безпосередньо та безпечно інтегрувати свої кастомні JSON `skill_pack.json` для винагородження специфічної поведінки (наприклад, купівля преміум-брендів), не змінюючи Python-логіку системи.

## 🛠 Як запустити

1. Встановіть залежності:

   ```bash
   pip install -r requirements.txt
   ```

2. Запустіть основне RPG-демо:

   ```bash
   python main.py
   ```

*(Або напряму конкретний режим через `streamlit run apps/web/streamlit_rpg/app.py` чи `apps/web/streamlit_b2b/app.py`)*

## 📊 KPI для пілотів

* **Залучення B2C**: ↑ Тривалість сесії, ↑ Повернення користувачів (Retention завдяки гейміфікації).
* **Ритейл B2B**: ↑ Середній чек (AOV), ↑ Додавання в кошик (ATC) повноцінних "образів" замість поодиноких речей, ↓ Повернення товарів.
